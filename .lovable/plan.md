

## Комментарии в полноэкранном попапе

### Что будет сделано

1. Кнопка "Прокомментировать" / "X комментариев" получит стрелку вправо (иконка `ChevronRight`) справа
2. Клик по кнопке открывает полноэкранный Dialog (используя существующий `Dialog` из ui) вместо инлайн-секции
3. Внутри попапа сверху -- содержимое поста (автор, текст, картинки, реакции), затем разделитель, затем комментарии или заглушка "Комментариев пока нет..."
4. Внизу попапа -- поле ввода нового комментария (прибито к низу)

### Визуальная структура попапа

```text
┌──────────────────────────────────────┐
│ [←] Комментарии                      │
├──────────────────────────────────────┤
│ [Аватар] Автор          · 2ч назад   │
│ Текст поста...                       │
│ [Картинки]                           │
│ [Реакции]                            │
├──────────────────────────────────────┤
│ Комментарии (3)                      │
│                                      │
│ [Аватар] Имя · 1ч назад             │
│   Текст комментария                  │
│ [Аватар] Имя · 30мин назад          │
│   Текст комментария                  │
│                                      │
│ --- или ---                          │
│ Комментариев пока нет...             │
│                                      │
├──────────────────────────────────────┤
│ [Поле ввода]              [Отправить]│
└──────────────────────────────────────┘
```

2. Заменить `commentsOpen` на `dialogOpen` для управления Dialog
3. Обновить кнопку комментариев: добавить `flex items-center justify-center gap-1` и иконку `ChevronRight` (h-4 w-4) справа от текста
4. Убрать инлайн-секцию комментариев (строки 200-258)
5. Добавить `Dialog` с `DialogContent` в стиле полного экрана:
   - `DialogContent` с классами: `max-w-full h-full sm:max-w-2xl sm:h-[90vh] flex flex-col p-0 gap-0`
   - Хедер: кнопка "назад" (стрелка влево) + заголовок "Комментарии"
   - Скроллируемая область: сначала пост (автор, текст, картинки, реакции), разделитель, затем список всех комментариев или "Комментариев пока нет..."
   - Прибитая к низу форма ввода комментария
6. Все комментарии показываются сразу в попапе (без "Показать ещё"), убрать `showAllComments` / `VISIBLE_COMMENTS` логику
