

## Добавить кнопку "Скрыть" для сворачивания L3

### Что будет сделано

Когда L3-пункты раскрыты через "Показать ещё", внизу списка появится кнопка "Скрыть", которая сворачивает их обратно до 3 пунктов.

### Технические детали

**Файл: `src/components/MobileCatalogMenu.tsx`**

В блоке рендеринга L3 (после `visibleL3.map(...)`) добавить условие: если список раскрыт (`isExpanded === true`) и пунктов больше 3, показать кнопку "Скрыть" с теми же стилями что и "Показать ещё" (`text-sm text-primary`, отступ `pl-12`). Клик по ней удаляет id L2-родителя из `expandedL3`.

Текущий код показывает "Показать ещё" только когда `hasMore && !isExpanded`. Нужно добавить второе условие: `hasMore && isExpanded` -- показать "Скрыть".
