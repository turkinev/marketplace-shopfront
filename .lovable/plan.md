

## Страница ленты постов (Telegram-стиль)

### Что будет сделано

Новая страница `/feed` с лентой постов в стиле Telegram-канала. Хедер (десктопный и мобильный) остаётся без изменений -- используется существующий `Layout`.

### Визуальная структура

```text
[Существующий хедер]
────────────────────────────────────────
        Лента новостей
────────────────────────────────────────
┌──────────────────────────────────────┐
│ [Аватар] Название канала   · 2ч назад│
│                                      │
│ Текст поста с **жирным**, ссылками   │
│ и обычным форматированием...         │
│                                      │
│ ┌──────────┐ ┌──────────┐            │
│ │  Фото 1  │ │  Фото 2  │            │
│ └──────────┘ └──────────┘            │
│                                      │
│ 👍 12  ❤️ 5  😂 3       💬 8        │
│                                      │
│ ── Комментарии (8) ──────────────────│
│ [Аватар] Имя · 1ч назад             │
│   Текст комментария                  │
│ [Аватар] Имя · 30мин назад          │
│   Текст комментария                  │
│ Показать ещё...                      │
│                                      │
│ [Поле ввода комментария]  [Отправить]│
└──────────────────────────────────────┘

┌──────────────────────────────────────┐
│ [Следующий пост...]                  │
└──────────────────────────────────────┘
```

### Компоненты

**1. `src/pages/Feed.tsx`** -- страница ленты
- Массив моковых постов с текстом, картинками, ссылками, реакциями и комментариями
- Рендерит список `PostCard` в контейнере `max-w-2xl mx-auto`
- Заголовок страницы "Лента новостей"

**2. `src/components/PostCard.tsx`** -- карточка поста
- Шапка: аватар автора, имя, время публикации (относительное через `date-fns`)
- Тело: текст с поддержкой переносов строк, жирного текста (`**text**`), и кликабельных ссылок (автодетект URL)
- Галерея картинок: сетка 1-4 изображений с адаптивной раскладкой
- Реакции: ряд кнопок-эмодзи (thumbs up, heart, fire, laugh и т.д.) с счётчиками, пользователь может ставить/убирать реакцию (toggle)
- Секция комментариев: разворачиваемая, первые 2 комментария видны, остальные за "Показать ещё"
- Форма добавления комментария: `Input` + кнопка отправки

**3. Маршрут в `src/App.tsx`**
- Добавить `<Route path="/feed" element={<Feed />} />`

### Технические детали

**Данные поста (типы)**:
```
Post { id, author: { name, avatar }, date, text, images?: string[], reactions: Reaction[], comments: Comment[] }
Reaction { emoji, label, count, isActive }
Comment { id, author: { name, avatar }, date, text }
```

**Форматирование текста**:
- Простой парсер: заменяет `**text**` на `<strong>`, URL на `<a>` с `target="_blank"`
- Сохраняет переносы строк через `whitespace-pre-line`

**Реакции**:
- Массив из 5-6 эмодзи (like, heart, fire, laugh, surprise, sad)
- Клик toggle: +1/-1 к счётчику, подсветка активной реакции (`bg-primary/10 border-primary`)
- Используются обычные символы эмодзи, не иконки

**Комментарии**:
- Показываются первые 2, "Показать ещё" раскрывает остальные
- Форма внизу: `Input` с placeholder "Написать комментарий..." и кнопка `Send`
- Новый комментарий добавляется в локальный стейт

**Стилизация** -- в едином стиле проекта:
- Карточки: `bg-card rounded-lg shadow-sm border border-border`
- Аватары: компонент `Avatar` из ui
- Кнопки: компонент `Button` из ui
- Цвета: `text-foreground`, `text-muted-foreground`, `bg-background`
- Отступы между постами: `space-y-4`
